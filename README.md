# 设备维度
## 配电箱

- 检查事项: 漏电保护测试
## 弱电箱

- 检查事项: 网线是否联通，插座是否正常
## 插座

- 检查事项: 零火线是否接对，地线是否连接，漏电保护测试，相应位置是否加装防水盒
## 网线接口

- 网线是否连接正确，见附录
## 空调

- 检查事项: 是否漏水，空调口是否固定整齐
## 油烟机

- 检查事项: 烟道是否接入烟井
## 凉霸

- 检查事项: 目前检查无反应(可能需要遥控)
## 厨房报警器

- 检查事项: 观察电源指示灯是否闪烁(正常),不闪烁(异常)
## 净水器

- 检查事项: 出水是否正常，是否漏水
## 马桶

- 检查事项: 冲水后是否持续漏水一段时间
## 水龙头

- 检查事项: 水龙头是否接地，是否漏水，水池下方是否有弯管,防止臭气上浮
# 协作
## 各方协作
### 共同利益

- 开发商、工程方、物业、业主之间并不冲突,大家都把自己的事情做好,达到共同利益
- 互相理解,不揣测、不人身攻击,对事不对人,不“要求”他人怎么做
### 交付心理

- 业主视角
   - 满足交房标准
      - 外在:外观/划痕/凹陷/墙体
      - 内在:水/电/气/烟道/空调水管/吊顶
      - 安全:漏水问题,电路漏电问题,接地问题
      - 设备:工作是否正常
   - 各项内容整改满意
   - 整改方式方法问题, 维修 or 替换
   - 整改期间物业费补贴问题
- 开发商视角
   - 服务:提升业主交付期间的各项服务,服务态度
   - 标准:标准化流程引导
   - 成本:最低成本整改(人力成本/物料成本)
   - 舆论:解决交付中各项问题,让客户满意,降低业主情绪
- 工程队视角
   - 快速:快速解决各项问题,小问题即刻修复
   - 整改:问题分类,统一提交各个负责人,分批次修复
   - 验收:整改完成后,统一验收,不符合标准的,重新整改,形成闭环
   - 主负责任人:积极协调各方分工负责人，统一把控全局，主人翁owner意识
   - 分工负责人:积极解决对应部分问题，准时高质量完成，并积极汇报总负责人
- 物业视角
   - 服务:服务好交付期间的所有人员
   - 流程:协调业主、工程房、开发商完成标准流程

- 商家关联方业主视角
   - 商家:推销改造,封窗,美缝,家具,软装,各自甄别
   - 半商家:自己选的不错的商品服务,推荐给同业主,建议
